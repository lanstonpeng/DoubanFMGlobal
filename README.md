#豆瓣FM的插件
###全局控制播放(在任意页面按)
- s    --> 下一首
- d    --> 垃圾桶
- f    --> 喜欢
- 空格 --> 暂停

###Note
- 请使用Vimium的同学进入insert mode后再操作(担心会影响你当前的工作状态)
- 多于一个douban.fm tab时会失效(我还不知道怎么判断一个tab是否播放还是暂停)

###How to use
- open extension managment,select "Developer mode"
- drag the github repo folder into the extension window
- reload your pages
